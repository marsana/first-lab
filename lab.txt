- Переключение между разделами страницы - простой самодельный скрипт на клик и скролл.

- Каждый раздел, если он меньше окна браузера, занимает целиком одно окно, если больше - скролится, а по окончанию, проскроливается сразу до следующего раздела. Пример: http://www.papertelevision.com/

- Перестраивание айтемов (masonry) - почему-то появился эффект дрожания (несколько секунд после окончания скролла)
  - для персонажей
  - для работ

- фильтрация айтемов по жанру - простой скрипт на работе с классами
(додумать до множественного выбора)

- открытие (magnific popup)
  - личных портфолио персонажей
  - описания работ

- доступность всех контентных разделов по url через hash

- анимация(перестраивание) айтемов при открытии внутренних разделов

- анимация логотипа (уменьшается и встаёт над меню) сайта при уходе с первого экрана

- становление одного из айтемов при открытом внешнем разделе кнопкой "закрыть"
